# Proyecto TuCondo - Base de Datos
Este repositorio contiene los scripts necesarios para desplegar la base de datos de Oracle SQL para el proyecto de Análisis de Sistemas.

## Instrucciones
1. Ejecutar el archivo `script_base_de_datos.sql`.
2. Utilizar la conexión de Tailscale para acceder al servidor central.
